Binary Calculator

Convert numbers between Base 2 and Decimal

How to Convert

Binary to Decimal

To convert from Base 2 to Base 10:

  1. Write down the binary number.
  2. Multiply each bit by 2 raised to the power of its position, starting from 0 on the far right.
  3. Add all the calculated values together to get the final decimal number.

Decimal to Binary

To convert from Base 10 to Base 2:

  1. Divide the decimal number by 2.
  2. Write down the remainder (which will be 0 or 1).
  3. Divide the resulting quotient by 2 again and write down the next remainder.
  4. Repeat this process until the quotient is 0. The binary number is all the remainders read in reverse order (bottom to top).
