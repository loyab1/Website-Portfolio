Python: The Automation Assistant

While HTML, CSS, and JavaScript run directly inside the user's web browser (the "Frontend"), Python is a powerful scripting language that we used behind the scenes (the "Backend" or Development Environment) to automate tedious tasks while building this project!

What is Python Used For Here?

Imagine needing to update the top navigation bar to add a new link. Doing it by hand means opening 8 different HTML files and carefully copying and pasting the code without making a mistake. Instead of doing it manually, we wrote Python scripts to act as our robotic assistant, automatically editing all 8 files in less than a second!

File I/O (Input / Output)

One of Python's greatest strengths is reading and writing files on the computer's hard drive.

with open("index.html", "r") as f:
  content = f.read()
# Modify the content...
with open("index.html", "w") as f:
  f.write(content)

In this code, we instruct Python to Open a file. The "r" stands for "Read Mode", which pulls the HTML code into a string variable. After we change the string, we open the file again in "w" ("Write Mode") and overwrite the old file with our new updates.

Libraries and Glob

Python has massive collections of pre-written code called Libraries that you can import to save time.

We used the glob library to easily find every single HTML file in our project folder simultaneously. By writing glob.glob("*.html"), Python instantly hands us a list containing index.html, boolean-logic.html, and the rest, so we can loop through them automatically.

Regular Expressions (Regex)

When we migrated our old AI Prompts log into a clean Markdown file, we needed to intelligently extract the dates, prompts, and tasks from the messy HTML tags. We did this using the re (Regex) library.

Regex is like the "Ctrl+F" search feature on steroids. We can write a mathematical pattern like <h3>(.*?)</h3>, which tells Python: "Find any text that is sandwiched between an h3 tag, pull it out, and save it." This allowed us to strip away the HTML and convert our logs into perfect text formatting!
