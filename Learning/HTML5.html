HTML5

HTML5 is the latest version of HTML. It is used to create web pages and applications. It is the starting point and most fundamental language to create a website.

Syntax

<!DOCTYPE html> - This tag defines the document is html. All HTML Documents must start with this declaration

<html> - The root element of a HTML page, all elements are defined with the openeing and closing of this tag.

    <head> - this tag contains meta information about the page like title scripts and style sheets elements that can be used within <head> are:

    • <meta> - Meta data such as specify character set, page description,keywords,author, and viewport settings
    • <title> - specifies the title for the page, content is shown in the title bar
    • <base>
    • <link> - relationship between current document and external resource
    • <style> - used to define style information for a single page
    • <script> - define client side javascripts
    • <template>

    </head>

    <body> - This tag defines the document body. A container for all visible contents. What the browser displays. Tags that are used within the body are usually:

      <p> - This tag creates a paragraph

      • A paragraph starts a new line, but the results of how it is displayed depends on the screen size
      • To separate content or create a thematic break use the <hr> tag

        • Example

      • the <pre> element is how you displayed fixed width text

      <h1> - This tag creates a large heading.

      • Headings are numbered h1-h6
      • Use only one <h1> per page, h2 section titles, h3 sub sections
      • Browsers will add white space before and after a heading
      • Heading size can be specified with css inline style attribute: style="font-size:60px;"

      <a> - This tag creates a link. Attributes like href need to be used

        HTML Link Target Attributes target=""

      • _self - Opens the document in the same tab it was clicked on
      • _blank - opens the document in a new window/tab
      • _parent - opens the document in the parent frame
      • _top - opens the document in the full body of the window.

      an Image can be used as a link, just include it within the <a> tag

      <img> - this Tag is is how images are defined. Attributes like src, alt, width, height are used

      <br> - This tag is used for a line break. Since it is an empty element it does not rquire a closing tag

      Comments can be written by writting text in between the follow syntax <!-- Text here -->

    </body>

    </html>

    Div Element

    A div element is used as a container for other HTML Elements. It takes up the full width available and has a line break before and after.

    There are no required attributes but <style>, <class>, and <id> are among the common.

    Example Div - here you can see it take up the full width.

    Attributes
    Class - often used to point to a class name in a style sheet and can be used on any HTML Element.
    id - used to point to a specific style declaration & specify a unique id for an HTML element
    Interactive Elements
    <button>
    A button tag creates a clickable button, an action must be added through java script. Buttons are often used inside forms. A button should always specify its type attribute.
    HTML Forms

    An HTML form is used to collect user input. Most often sent to a server for processing. A form is created using the <form> tag.

    The element itself is a container for different types such as text fields, checkboxes, radio buttons, submit buttons and many more.
